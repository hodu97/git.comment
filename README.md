#github에 코드 업로드 하기 🥦

1. 초기화

   ```
  git init
  ```
# .git이라는 폴더가 생성

2. 파일올리기
```
git add .
```
3. 히스토리 만들기
```
git commit -m "내가 적고싶은 메세지 적으면 됨"
```
